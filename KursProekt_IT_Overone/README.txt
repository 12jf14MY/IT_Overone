Идея:
	Идея заключалась в том, чтобы создать что-то громоздкое, большое и непонятное, запутать всех и в это же время самому разобраться с Pyqt5 более 
	углубленно и начать собой гордиться что "я могу". Для Django еще "слабак" так что решил все делать в Pyqt5 здесь логика немного понятна
	хоть и геморно прописывать каждые кнопочки и загагулечки.

Цель проекта:

	Получение и закрепление нужных знаний в PyQt5. А самое главное получить сертификат об обучении и отличном прохождении курсов, который мне
	почему-то уверен в будущем не понадобится. Зачем мне это все? - Не знаю - сам долблю себя этим вопросом. Точнее все я знаю, но не скажу.

Реализация:

	Программа реализована в виде 6 модулей, main-главный и 5 проектов дополнительных, в модуле main как и 
	полагается лежит основной код проекта, а в модулях остальных прописаны 5 самостоятельных отдельных кода со своими окнами, 
	которые запускаются при нажатии на кнопки в главном модуле.
	Все иконки программы находятся в своих папках. 
	В каждом проекте свои классы главных окон которые разделены на функции по назначению.

Технологии и библиотеки:

	обязательно нужно установить pip install PyQt5, иначе ничего работать не будет

	#Mywebbrowser
	Нужно установить pip install QtwebEngine.
	Это  копия веб-браузера, сделанная с использованием pyqt5 и Qtwebengine с поисковой системой google.

	#MYpaint 
	Аналог  виндовсоского paint.

	#QrCode
	нужно установить pip install qrcode
	Cоздатель Qr кода из любого текста

	#calc
	Обычный калькулятор для выполнения стандартных математических операций 

	#notepad
	Окно для заметок. Запись, сохранение, редактирование любое действие с текстом.
